# Website Authentication dengan Notifikasi Telegram

Sistem website lengkap dengan fitur login, register, dashboard, dan notifikasi Telegram real-time.

## Fitur Utama

1. **Halaman Login** - Form login dengan validasi
2. **Halaman Register** - Form pendaftaran dengan validasi dan pengecekan kekuatan password
3. **Dashboard** - Tampilan statistik dan aktivitas pengguna
4. **Notifikasi Telegram** - Kirim notifikasi ke bot Telegram setiap ada aktivitas login/register
5. **Database Lokal** - Penyimpanan data menggunakan localStorage
6. **Responsive Design** - Tampilan yang optimal di semua perangkat
